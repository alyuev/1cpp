# 1С++ (icpp) — внешняя компонента для 1С:Предприятие 7.7

**1С++** — свободная внешняя компонента (VK) для **1С:Предприятие 7.7**, которая
существенно расширяет встроенный язык 1С:

- **ООП** — собственные классы, наследование, полиморфизм, интерфейсы;
- **обработка исключений** (`Попытка … Исключение … КонецПопытки`);
- **дополнительные типы** — индексированная таблица, хеш-таблица, расширенный
  список значений, структура, коллекции и т.д.;
- **прямой доступ к данным** — SQL через ODBC/OLEDB и прямые запросы к БД 1С;
- **доступ к MFC/ATL/COM**, метаданным конфигурации, событиям и многое другое.

Компонента загружается штатно: `ЗагрузитьВнешнююКомпоненту(КаталогИБ() + "1cpp.dll")`.

## Про этот репозиторий

Здесь — исходники 1С++ **версии 3.2.4.3** и **воспроизведённая сборка** готовой
`1cpp.dll` на оригинальном тулчейне проекта (Intel C++ Compiler). Результат —
рабочая **самодостаточная** компонента, идентичная по поведению эталонной:
зависит только от `MFC42.DLL`, `MSVCRT.dll` и рантайма 1С (**без `libmmd.dll`**).

> ✅ Проверено загрузкой в 1С 7.7: `ЗагрузитьВнешнююКомпоненту = 1`, создаются
> классы 1С++ (`HashMap`, `MetaInfoClasses`, `ИндексированнаяТаблица`).
> Тест — база [`TestBase/`](TestBase/) (`run_base.ps1`).

## Скачать готовую компоненту

Последний собранный `1cpp.dll` — в разделе
**[Releases](https://github.com/alyuev/1cpp/releases/latest)**
(прямая ссылка на бинарник) либо в репозитории: [`dist/1CPP.dll`](dist/1CPP.dll).

## Сборка из исходников

Тулчейн (форумный рецепт): **ICL 11.1.054 + PSDK2003 MFC 6.0 (class CString) +
VC6 CRT (msvcrt) + STLport 5.2.1 (/MD) + Boost 1.34.1 + forwarder**.

```powershell
build_icl.ps1 -Stage all -Vc6 <путь к VC6 MSVC600>
```

Подготовленный STLport уже лежит в репозитории (`deps/stlport_icl/`); из внешнего
нужен только **ванильный Boost 1.34.1** и установленный тулчейн — см.
[**DEPS.md**](DEPS.md). Полный рецепт сборки, все правки исходников и ключевые
нюансы (DllMain из MFC, `/DEF` через `link.exe`, статический рантайм Intel) —
в [**СБОРКА_форум.md**](СБОРКА_форум.md).

## Структура

| Каталог | Что |
|---|---|
| `Source/` | исходники компоненты (+ `Source/LIBS` — import-либы 1С, `Source/Forwarder` — мост wchar_t) |
| `Testing/` | набор юнит-тестов 1С++ |
| `deps/stlport_icl/` | подготовленный STLport (патченные заголовки + собранная либа) |
| `dist/` | готовая `1CPP.dll` |
| `TestBase/` | пустая тест-база 1С для проверки загрузки компоненты |
| `build_icl.ps1` | скрипт сборки на ICL |
